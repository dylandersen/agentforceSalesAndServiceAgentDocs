## Key Capabilities

**Record Creation with Lightning Types**

Create Accounts, Opportunities, Cases, and Contacts through conversational AI with custom Lightning Type UI components featuring:
- Custom input editors with SLDS2 card interfaces
- Dynamic picklist loading from org configuration
- Professional result renderers with formatted output
- Real-time validation and help text

**AI-Powered Intelligence**

Comprehensive analysis capabilities including:
- **Account Intelligence**: 6-dimension health scoring with Overall Health Score (0-100)
- **Opportunity Analysis**: Sales guidance with activity timeline and deal momentum
- **Case Priority Analysis**: AI-powered priority scoring and recommendations
- **Similarity Matching**: Semantic matching for opportunities and cases

**Smart Field Management**

Update and manage record fields with confidence:
- Confirmation-based field updates
- Undo capability for all changes
- Support for all field types
- Audit trail maintenance

**Custom Lightning Types**

Rich UI components for data collection and display:
- Custom input editors for structured data
- Beautiful output renderers with card layouts
- Type safety through JSON schema validation
- Reusable patterns across all functions
