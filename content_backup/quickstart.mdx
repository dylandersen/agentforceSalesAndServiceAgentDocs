## Installation

<Steps>
  <Step>
    **Clone or Download**
    
    Get the project files from your repository:
    
    ```bash
    git clone <repository-url>
    cd agentforceSalesAndServiceAgent
    ```
  </Step>

  <Step>
    **Deploy Apex Classes**
    
    Deploy all Apex classes and test classes first:
    
    ```bash
    sf project deploy start --source-dir force-app/main/default/classes
    ```
    
    <Callout type="info">
      Apex classes must be deployed before Lightning Types that reference them.
    </Callout>
  </Step>

  <Step>
    **Deploy Lightning Web Components & Lightning Types**
    
    Deploy LWCs and Lightning Types together in a single deployment:
    
    ```bash
    sf project deploy start --source-dir force-app/main/default/lwc,force-app/main/default/lightningTypes
    ```
    
    <Callout type="warn">
      Lightning Type bundles reference LWCs. They must be deployed together to ensure LWC references resolve correctly.
    </Callout>
  </Step>

  <Step>
    **Deploy Flows and GenAI Components**
    
    Deploy the orchestration and AI components:
    
    ```bash
    sf project deploy start --source-dir force-app/main/default/flows
    sf project deploy start --source-dir force-app/main/default/genAiFunctions
    sf project deploy start --source-dir force-app/main/default/genAiPlugins
    ```
  </Step>

  <Step>
    **Configure Field-Level Security**
    
    Set up field permissions for System Administrator and Standard User profiles:
    
    - Navigate to **Setup â†’ Object Manager â†’ [Object] â†’ Fields**
    - For each custom field, set:
      - System Administrator: Create/Read/Edit/Delete
      - Standard User: Create/Read/Edit/Delete
  </Step>

  <Step>
    **Configure Your Agent**
    
    Add the GenAI Plugins to your Agentforce agent:
    
    1. Go to **Setup â†’ Agents**
    2. Select or create an agent
    3. Add these plugins:
       - Create Account with Agentforce
       - Create Opportunities with Agentforce
       - Create Case with Agentforce
       - Create Contact with Agentforce
       - Account Intelligence
       - Opportunity Intelligence
       - Priority Intelligence
  </Step>
</Steps>

## Quick Usage Examples

### Create an Account

<Steps>
  <Step>
    **Start Conversation**: "Create a new account"
  </Step>
  <Step>
    **Provide Details**: Fill in the custom Lightning Type form with:
    - Name (required)
    - Phone (required)
    - Website (required)
    - Industry (picklist)
    - Type (picklist)
    - Billing Address (optional)
  </Step>
  <Step>
    **Submit**: Agent creates the account and displays formatted results with navigation link
  </Step>
</Steps>

### Analyze Account Health

### Create an Opportunity

<Steps>
  <Step>
    **Initiate**: "Create a new opportunity for Acme Corp"
  </Step>
  <Step>
    **Fill Form**:
    - Account: Acme Corporation
    - Opportunity Name: Q4 Software License Renewal
    - Amount: $50,000
    - Stage: Negotiation/Review (from dynamic picklist)
    - Description: Expansion by 20%
  </Step>
  <Step>
    **Confirm**: Agent confirms details and creates the opportunity
  </Step>
</Steps>
