## What is Agentforce?

A comprehensive Salesforce Agentforce implementation that brings **conversational AI** to your record management, intelligent analysis, and field automation workflows. Built with modern Lightning Types for a beautiful, type-safe UI experience.

<Callout type="info" title="Version 0.5 Now Available">
  **New**: Account Relationship Health Dashboard with real-time monitoring, enhanced AI analysis with specific data citations, and rich HTML output formatting.
</Callout>

## Why Choose Agentforce?

### Hybrid Architecture Design

Built on four powerful layers working seamlessly together:

- **Apex Classes** - Complex business logic and AI integration
- **Flows** - Declarative orchestration for easy configuration
- **GenAI Functions** - Natural language interfaces with schema validation
- **Lightning Types** - Beautiful custom UI components

## Quick Start

<Steps>
  <Step>
    **Prerequisites**: Ensure Salesforce Enterprise Edition or higher with Agentforce and Einstein AI enabled
  </Step>
  
  <Step>
    **Deploy**: Use Salesforce CLI to deploy Apex classes, Lightning Types, Flows, and GenAI components
  </Step>
  
  <Step>
    **Configure**: Set up field-level security and configure your Agentforce agent with the provided plugins
  </Step>
  
  <Step>
    **Launch**: Start creating records and analyzing data through natural conversational AI
  </Step>
</Steps>
