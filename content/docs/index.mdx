## Welcome to Agentforce Sales & Service Agent

A comprehensive Salesforce Agentforce implementation providing AI-powered record creation, intelligent analysis, similarity matching, and field management with custom Lightning Types.

### Key Features

**AI-Powered Record Creation**
Create Accounts, Opportunities, Cases, and Contacts through conversational AI with beautiful custom Lightning Type UI components.

**Intelligence & Analysis**
- 6-dimension account health scoring
- Opportunity sales guidance with activity timelines
- Case priority analysis with AI recommendations
- Semantic similarity matching for opportunities and cases

**Smart Field Management**
Update and manage record fields with confirmation-based updates, undo capability, and full audit trail maintenance.

**Custom Lightning Types**
Rich UI components providing custom input editors, professional result renderers, type-safe JSON schema validation, and reusable patterns.

## Getting Started

1. **Prerequisites**: Ensure you have Salesforce Enterprise Edition or higher with Agentforce and Einstein AI enabled
2. **Deploy**: Use Salesforce CLI to deploy Apex classes, Lightning Types, Flows, and GenAI components
3. **Configure**: Set up field-level security and configure your Agentforce agent
4. **Use**: Start creating records and analyzing data through conversational AI
